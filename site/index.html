Execution surface

같은 source를 더 많이 붙이는 것보다, 더 정확히 보내는 쪽에 집중합니다.

MCP + CLI

OpenClaw, Claude Desktop, Cursor 같은 MCP client에서 쓰고, 로컬에서는 `kgab` CLI로 같은 surface를 바로 재현합니다.

Source-first trust

모든 응답은 원문 URL, identifier, fetched_at를 포함하고, provider raw shape 대신 질문 중심 normalized shape로 정리됩니다.

Routing

resolver는 이제 세 가지 상태로 다음 행동을 분명히 말합니다.

ready

바로 실행 가능

`행정기본법 제1조` 같은 질의는 법령명과 조문을 추출해서 바로 `get_law_text`로 넘깁니다.

needs_input

필수값이 더 필요함

비교 대상은 맞지만 기간이 없으면 `start`, `end`만 추가로 물어보고 실행 템플릿을 함께 냅니다.

needs_disambiguation

해석 축을 좁혀야 함

`민간위탁 현황`처럼 애매한 질문은 law, bill, dataset 중 어느 방향인지 옵션으로 정리합니다.
